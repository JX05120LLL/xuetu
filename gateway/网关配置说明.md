# 网关配置说明

## 🔧 配置模式

### 1. 本地开发模式（推荐）
使用 `application-local.yml` 配置，不依赖Nacos，直接连接本地服务。

**启动命令**：
```bash
java -jar gateway.jar --spring.profiles.active=local
```

或在IDE中设置环境变量：
```
spring.profiles.active=local
```

### 2. 生产模式
使用 `application.yml` 配置，通过Nacos进行服务发现。

**启动命令**：
```bash
java -jar gateway.jar
```

## 🐛 常见问题

### 1. 404 Not Found
**原因**：网关路由配置错误或后端服务未启动

**解决**：
1. 检查后端服务是否启动：`netstat -ano | findstr "8077"`
2. 检查网关日志，查看路由匹配情况
3. 确认请求路径与路由配置一致

### 2. 503 Service Unavailable
**原因**：后端服务未启动或网关无法连接

**解决**：
1. 确认后端服务已启动
2. 使用本地模式：`--spring.profiles.active=local`
3. 检查端口是否被占用

### 3. 500 Internal Server Error
**原因**：后端服务异常

**解决**：
1. 查看后端服务日志
2. 检查数据库连接
3. 验证请求参数格式

### 4. CORS跨域问题
**原因**：跨域配置未生效

**解决**：
1. 网关已配置全局CORS，应该不会有问题
2. 如仍有问题，检查后端服务是否也配置了CORS（可能冲突）
