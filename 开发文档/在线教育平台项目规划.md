# 在线教育平台项目规划（微服务架构）

## 一、项目概述

### 1.1 项目目标
构建一个基于微服务架构的在线教育平台，包含用户前台系统和后台管理系统，实现课程展示、购买和学习等功能。该项目旨在展示Java后端开发能力和微服务架构理解，作为寻找实习职位的简历项目。

### 1.2 技术栈
- **后端**：Java 11+, Spring Boot 2.7.x, Spring Cloud 2021.x
- **微服务组件**：Spring Cloud Gateway, Nacos, OpenFeign, Sentinel
- **数据存储**：MySQL 8.0, Redis 6.x
- **认证授权**：JWT + Token
- **前端**：Vue.js 3.x, Element Plus (可选实现)

```java
// JWT工具类示例
@Component
public class JwtUtils {
    @Value("${jwt.secret}")
    private String secret;

    @Value("${jwt.expire}")
    private long expire;

    public String generateToken(Long userId) {
        Date now = new Date();
        Date expireDate = new Date(now.getTime() + expire * 1000);

        return Jwts.builder()
                .setSubject(userId.toString())
                .setIssuedAt(now)
                .setExpiration(expireDate)
                .signWith(SignatureAlgorithm.HS512, secret)
                .compact();
    }

    public Long getUserIdFromToken(String token) {
        Claims claims = Jwts.parser()
                .setSigningKey(secret)
                .parseClaimsJws(token)
                .getBody();
        return Long.parseLong(claims.getSubject());
    }
}
```

```java
// 订单创建示例
@Service
public class OrderService {
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Autowired
    private CourseClient courseClient;

    public Order createOrder(Long userId, Long courseId) {
        // 分布式锁防止超卖
        String lockKey = "lock:course:" + courseId;
        Boolean acquired = redisTemplate.opsForValue().setIfAbsent(lockKey, "1", 10, TimeUnit.SECONDS);
        
        if (Boolean.TRUE.equals(acquired)) {
            try {
                // 创建订单逻辑
                Order order = new Order();
                // ... 订单创建代码 ...
                return order;
            } finally {
                // 释放锁
                redisTemplate.delete(lockKey);
            }
        } else {
            throw new BusinessException("下单失败，请稍后重试");
        }
    }
}
```
