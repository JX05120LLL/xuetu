## 🛠️ 技术栈选择

### 后端技术栈（主攻方向）
```yaml
核心框架: Spring Boot 2.7 (学习成本低)
数据库: MySQL 8.0 (最常用)
缓存: Redis (基础用法)
认证: JWT (相对简单)
API文档: Knife4j (自动生成)
微服务: 
  - 用户服务 (user-service)
  - 课程服务 (course-service) 
  - 订单服务 (order-service)
  - 学习服务 (learning-service)
  - AI服务 (ai-service) # 重点突破
```

### 前端技术栈（次要）
```yaml
框架: Vue 3 + Element Plus (学习资源多)
状态管理: Pinia (比Vuex简单)
HTTP客户端: Axios (标准选择)
图表: ECharts (成熟易用)
```

### AI技术栈（重点亮点）
```yaml
大模型API: 
  - 百度文心一言 (国内访问稳定)
  - 讯飞星火 (教育场景友好)
  - 智谱GLM (性价比高)
向量搜索: 简化实现，用MySQL存储
推荐算法: 协同过滤的简单实现
```

## 🚀 项目部署方案（学生版）

### 推荐部署架构
```
用户访问
    ↓
域名 (xuetu-ai.com)
    ↓  
阿里云学生服务器 (2核4G)
├── 前端 (Nginx)
├── 后端服务 (Spring Boot)
├── 数据库 (MySQL)
└── 缓存 (Redis)
```

### 部署步骤
1. **购买服务器**: 阿里云学生机 (最便宜选择)
2. **环境配置**: Docker + JDK + Node.js
3. **数据库搭建**: MySQL + Redis
4. **应用部署**: 
   ```bash
   # 后端部署
   nohup java -jar xuetu-platform.jar &
   
   # 前端部署  
   npm run build
   # 上传到nginx目录
   ```
5. **域名解析**: 绑定域名到服务器IP
6. **SSL证书**: 免费证书配置HTTPS

### 监控方案
- **简单监控**: 系统自带监控
- **日志管理**: 文件日志 + 定期清理
- **备份策略**: 定期数据库备份
