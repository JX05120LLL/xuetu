## 1. 角色权限实现方式

### 1.1 整体架构（RBAC - 基于角色的访问控制）

```
用户 (User) → 角色 (Role) → 权限 (Permission)
     1:N           N:N            N:1
                                  ↓
                            接口/资源 (API/Resource)
```

**工作流程**：
1. 用户登录 → 查询用户的角色
2. 根据角色 → 查询角色的权限
3. 访问接口 → 验证是否有权限

### 1.2 实现层次

#### ✅ 方案A: 网关统一拦截（推荐）⭐⭐⭐⭐⭐

**优势**: 
- 统一管理，一处配置
- 所有服务自动受保护
- 性能高，拦截早

**实现位置**: `gateway` 模块

```java
// Gateway中的权限过滤器
@Component
public class AuthorizationFilter implements GlobalFilter {
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        // 1. 从JWT Token中获取用户角色
        String token = getToken(exchange);
        List<String> roles = parseRolesFromToken(token);
        
        // 2. 获取请求路径
        String path = exchange.getRequest().getPath().value();
        
        // 3. 检查是否有权限访问
        if (!hasPermission(roles, path)) {
            return unauthorized(exchange);
        }
        
        return chain.filter(exchange);
    }
}
```

#### ✅ 方案B: 各服务独立验证

**优势**: 
- 服务自主控制
- 灵活度高
- 可以做数据级权限

**实现位置**: 各个业务服务

```java
// 在各个Controller中使用注解
@PreAuthorize("hasRole('ADMIN')")
@PostMapping("/course/create")
public R<Course> createCourse(...) {
    // ...
}
```

#### ⭐ 推荐：混合方式

- **网关层**: 基础权限验证（是否登录、角色基本权限）
- **服务层**: 细粒度权限验证（具体功能权限、数据权限）

// UserServiceImpl.java
public LoginResponse login(LoginRequest request) {
    // 1. 验证用户名密码
    User user = userMapper.selectByUsername(request.getUsername());
    if (user == null || !passwordUtil.matches(request.getPassword(), user.getPassword())) {
        throw new UserAuthException("用户名或密码错误");
    }
    
    // 2. 🆕 查询用户角色
    List<Role> roles = roleMapper.selectUserRoles(user.getId());
    List<String> roleNames = roles.stream()
        .map(Role::getRoleName)
        .collect(Collectors.toList());
    
    // 3. 🆕 查询角色权限
    List<String> permissions = permissionMapper.selectUserPermissions(user.getId());
    
    // 4. 🆕 生成包含角色信息的JWT Token
    Map<String, Object> claims = new HashMap<>();
    claims.put("userId", user.getId());
    claims.put("username", user.getUsername());
    claims.put("roles", roleNames);  // 🆕 角色列表
    claims.put("permissions", permissions);  // 🆕 权限列表
    
    String token = jwtUtil.generateToken(claims);
    
    // 5. 返回登录信息
    return LoginResponse.builder()
        .token(token)
        .userId(user.getId())
        .username(user.getUsername())
        .roles(roleNames)  // 🆕 返回角色
        .permissions(permissions)  // 🆕 返回权限
        .build();
}

### 第4步: Gateway 添加权限验证过滤器 ⭐⭐⭐⭐⭐

```java
// gateway/filter/AuthorizationFilter.java
@Component
@Order(2)  // 在JWT验证之后
public class AuthorizationFilter implements GlobalFilter {
    
    @Autowired
    private JwtUtil jwtUtil;
    
    // URL权限配置
    private static final Map<String, List<String>> URL_PERMISSIONS = new HashMap<>();
    
    static {
        // 课程管理接口 - 需要讲师或管理员
        URL_PERMISSIONS.put("/api/course/create", Arrays.asList("TEACHER", "ADMIN"));
        URL_PERMISSIONS.put("/api/course/update", Arrays.asList("TEACHER", "ADMIN"));
        URL_PERMISSIONS.put("/api/course/delete", Arrays.asList("TEACHER", "ADMIN"));
        
        // 订单管理 - 需要运营或管理员
        URL_PERMISSIONS.put("/api/order/all", Arrays.asList("ADMIN", "OPERATOR"));
        URL_PERMISSIONS.put("/api/order/refund", Arrays.asList("ADMIN", "OPERATOR"));
        
        // 用户管理 - 只有管理员
        URL_PERMISSIONS.put("/api/user/list", Arrays.asList("ADMIN"));
        URL_PERMISSIONS.put("/api/role/**", Arrays.asList("ADMIN"));
    }
    
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        String path = exchange.getRequest().getPath().value();
        
        // 1. 从请求头获取Token
        String token = exchange.getRequest().getHeaders().getFirst("Authorization");
        if (token == null || !token.startsWith("Bearer ")) {
            return unauthorized(exchange);
        }
        
        token = token.substring(7);
        
        // 2. 解析Token，获取角色
        try {
            Claims claims = jwtUtil.parseToken(token);
            List<String> userRoles = (List<String>) claims.get("roles");
            
            // 3. 检查是否有权限
            List<String> requiredRoles = getRequiredRoles(path);
            if (requiredRoles != null && !hasAnyRole(userRoles, requiredRoles)) {
                return forbidden(exchange);
            }
            
            // 4. 将用户信息传递给下游服务
            ServerHttpRequest request = exchange.getRequest().mutate()
                .header("X-User-Id", claims.get("userId").toString())
                .header("X-User-Roles", String.join(",", userRoles))
                .build();
            
            return chain.filter(exchange.mutate().request(request).build());
            
        } catch (Exception e) {
            return unauthorized(exchange);
        }
    }
    
    private List<String> getRequiredRoles(String path) {
        for (Map.Entry<String, List<String>> entry : URL_PERMISSIONS.entrySet()) {
            if (path.matches(entry.getKey().replace("**", ".*"))) {
                return entry.getValue();
            }
        }
        return null;  // 不需要特殊权限
    }
    
    private boolean hasAnyRole(List<String> userRoles, List<String> requiredRoles) {
        return userRoles.stream().anyMatch(requiredRoles::contains);
    }
}
```

    // 如果不是管理员，只能查自己的
    if (!roles.contains("ADMIN") && !userId.equals(currentUserId)) {
        throw new UnauthorizedException("无权查看他人学习记录");
    }
    
    return learningRecordMapper.selectByUserId(userId);
}
