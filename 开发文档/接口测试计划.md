## 🔄 业务流程梳理

### 1️⃣ 用户注册登录流程
```
1. 用户注册 → POST /api/user/auth/register
   ├── 检查用户名 → GET /api/user/auth/check-username
   ├── 检查邮箱 → GET /api/user/auth/check-email
   └── 创建用户账号

2. 用户登录 → POST /api/user/auth/login
   ├── 验证用户名密码
   └── 返回JWT Token

3. 角色分配 (管理员操作)
   ├── 创建角色 → POST /api/admin/role/create
   ├── 为角色授予权限 → POST /api/admin/role/grant
   └── 为用户分配角色 → POST /api/admin/role/assign
```

### 2️⃣ 课程浏览购买流程
```
1. 浏览课程分类 → GET /api/course/category/tree
   
2. 查看课程列表 → GET /api/course/course/list
   ├── 按分类筛选
   ├── 按难度筛选
   └── 查看热门课程 → GET /api/course/course/hot

3. 查看课程详情 → GET /api/course/course/{id}
   ├── 查看章节列表 → GET /api/course/chapter/course/{courseId}
   ├── 查看课程评论 → GET /api/course/comment/course/{courseId}
   └── 查看评论统计 → GET /api/course/comment/course/{courseId}/statistics

4. 创建订单 → POST /api/order/api/orders
   ├── 检查是否可购买 → POST /api/order/api/orders/check-purchasable
   └── 返回订单信息

5. 支付订单 → POST /api/order/api/payments
   ├── 创建支付记录
   └── 模拟支付 → POST /api/order/api/payments/{paymentNo}/simulate
   └── (或快捷支付) → POST /api/order/api/payments/quick-pay
```

### 3️⃣ 学习流程
```
1. 开始学习课程
   ├── 查看课程章节 → GET /api/course/chapter/course/{courseId}
   └── 查看课程学习进度 → GET /api/learning/learning/record/progress/{courseId}

2. 记录学习进度 → POST /api/learning/learning/record/progress
   ├── 更新播放进度
   └── 标记课时完成 → POST /api/learning/learning/record/complete

3. 创建学习笔记 → POST /api/learning/learning/note
   ├── 查看课程笔记 → GET /api/learning/learning/note/course/{courseId}
   └── 搜索笔记 → GET /api/learning/learning/note/search

4. 发表评论 → POST /api/course/comment
   ├── 点赞评论 → POST /api/course/comment/{commentId}/like
   └── 回复评论 → POST /api/course/comment/reply
```

### 4️⃣ AI功能流程
```
1. AI问答 → POST /api/ai/chat/ask
   ├── 查看对话历史 → GET /api/ai/chat/history
   └── 清空对话 → DELETE /api/ai/chat/clear

2. 课程推荐 → GET /api/ai/recommend/courses
   └── 学习路径规划 → POST /api/ai/recommend/path

3. 学习分析 → GET /api/ai/analysis/report
   └── 获取学习建议 → GET /api/ai/analysis/advice
```
