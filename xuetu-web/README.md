# 学途在线教育平台 - 前端项目

## 项目介绍
学途在线教育平台是一个综合性的在线教育系统，提供课程学习、在线支付、笔记记录、AI助手等功能。本仓库包含平台的前端代码，采用Vue 3 + TypeScript + Vite + Element Plus开发。

## 功能特性

### 用户系统
- **用户认证**: 注册、登录、找回密码
- **个人中心**: 用户资料管理、头像上传
- **个人设置**: 修改密码、个人信息设置

### 课程系统
- **课程浏览**: 分类筛选、关键词搜索
- **课程详情**: 课程介绍、章节列表、用户评价
- **课程学习**: 视频播放、进度保存、笔记记录

### 购物与支付
- **购物车**: 添加课程、结算
- **订单管理**: 订单列表、订单详情、取消订单

### 笔记系统
- **创建笔记**: 课程学习时记笔记
- **笔记管理**: 查看、编辑、删除笔记

### AI助手
- **智能推荐**: 基于学习记录推荐课程
- **学习分析**: 分析学习数据，提供学习建议

## 技术栈

- **前端框架**: Vue 3
- **开发语言**: TypeScript
- **构建工具**: Vite
- **UI组件**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios
- **CSS预处理**: SCSS

## 开发环境搭建

### 前提条件
- Node.js v16+
- npm v8+ 或 yarn v1.22+

### 安装依赖
```bash
npm install
```

### 开发服务器启动
```bash
npm run dev
```

### 生产环境构建
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 主要页面

### 用户系统
- **个人中心**: 展示用户基本信息、学习统计、最近学习课程
- **我的课程**: 用户已购买的课程列表，支持筛选
- **我的笔记**: 用户学习笔记管理，支持搜索和编辑
- **个人设置**: 用户资料维护，头像上传，密码修改

### 讲师系统
- **成为讲师**: 讲师申请表单，讲师权益和申请流程展示

### 课程浏览与学习
- **课程列表**: 分类展示所有课程，支持筛选和搜索
- **课程详情**: 展示课程详细信息，评价和章节
- **视频播放**: 课程视频播放界面，支持笔记记录

### 订单与购物
- **购物车**: 管理待购课程
- **订单确认**: 确认订单信息
- **订单列表**: 查看历史订单
- **订单详情**: 查看订单详细信息

## 特色功能

### 1. 骨架屏加载
提升用户体验，在数据加载过程中显示骨架屏

### 2. 响应式设计
适配各种屏幕尺寸，包括移动设备

### 3. 统一状态管理
使用Pinia管理全局状态，分离UI和业务逻辑

### 4. 用户友好的空状态
针对空数据提供友好的提示和引导

### 5. 模块化组件设计
通过组件化提高代码复用性和可维护性

## API接口
前端通过API与后端服务交互，所有API路径以`/api`为前缀，由Nginx反向代理到后端网关。

## 环境变量
- `VITE_API_BASE_URL`: API服务器地址，默认为`http://localhost:8080`

## 部署注意事项
1. 确保Nginx配置支持Vue Router的history模式
2. 根据实际环境修改API基础URL
3. 静态资源应配置适当的缓存策略
